# Opacity-Project Front-end

A internal front-end project to Opacity-Project by **Akatsuki JS**

### :construction: STATUS: work in progress

Libs utilized:

- Flavor: flow
- Lib: React
- Formatter: prettier-standard
- Linter: standard
- Bundler: Webpack 4
- Git-hooks: husky
- Styles: Emotion JS

##  :book: Scripts

#### Install Dependencies

```shell
yarn install
```

#### Running
**1. Development**
```shell
yarn dev
```

**2. Production**

```shell
yarn prod
```

#### Typecheck
```shell
yarn flow
```

#### Prettier and Linter
**1. Linter**
```shell
yarn lint:fix
```

**2. Prettier + Linter**
```shell
yarn prettier
```

#### Storybook
```shell
yarn storybook
```
